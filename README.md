Currency Converter

A small Python project that converts currencies using the [Frankfurter.dev](https://www.frankfurter.dev) API.

---

Functionality

- Gets current exchange rates via API
- Saves the result to `JSON` and `CSV` files  
- Handles errors (incorrect input, network issues, etc.)

---

Installation

1. Clone the repository:
   git clone https://github.com/Marlay-0845/Currency_Converter.git
   cd currency-converter

2. Install dependencies:
    pip install -r requirements.txt
